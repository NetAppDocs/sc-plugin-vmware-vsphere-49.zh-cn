= 修改配置默认值
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
要提高运营效率、您可以修改 `scbr.override` 用于更改默认值的配置文件。这些值控制备份期间创建或删除的 VMware 快照数量或备份脚本停止运行之前的时间等设置。

。 `scbr.override` 适用于VMware vSphere的SnapCenter 插件在支持基于SnapCenter 应用程序的数据保护操作的环境中使用配置文件。如果此文件不存在，则必须从模板文件创建它。
